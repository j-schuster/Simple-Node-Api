JavaScript Questions API

An API for JavaScript Questions

This is an API for a currently work in progress React Native Application that allows JavaScript developers test their knowledge about a variety of JavaScript topics.

The tool will provide feedback on a developer's knowledge and expose the areas that need study. The app will allow a user to take a timed test according to a specific experince level and technology, and optionally compete against other developers.

All developers are welcome to contribute to the project, either by helping to build the app, or by extending the questions.
