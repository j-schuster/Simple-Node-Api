JavaScript Questions API

This is an API for a currently work in progress React Native Application that allows JavaScript developers test their knowledge about a variety of JavaScript topics.

The tool will provide feedback on a developer's knowledge and expose the areas that need study. The app will allow a user to take a timed test according to a specific experince level and technology, and optionally compete against other developers.

All developers are welcome to contribute to the project, either by helping to build the app, or by extending the questions.

If you're interested in contributing to the questions, make a post request from any browser to the following URL:

https://js-questions-api.herokuapp.com/api/questions

Each post must be an object that includes the fields from the following example.

$.ajax({
        type: "POST",   
        url: url,
        data: data,
        success: success,
        dataType: dataType
      });
 
const data = {
       title: 'What is a variable?',
       category: 'Vanilla Js',
       answers: ['a function','something that stores a value','a method','an api'],
       correctAnswer: 'something that stores a value',
       level: 'beginner'
     }

Categories: React, Vanilla-js, Redux, Node, Angular, Functional Programming, es6+

Answers: must be an array consisting of 4 answers, 3 incorrect and 1 correct.
